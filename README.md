Aplikasi pemecah uang Rupiah (Rp) menjadi pecahan-pecahan Rp. 100.000, RP. 50.000, 20.000, Rp. 10.000, Rp. 5.000, Rp. 1.000, Rp. 500, Rp. 100 dan Rp. 50.

## Quick Start
``` bash
yarn
yarn dev
```
Kemudian akses <http://localhost:3000> di browser.
